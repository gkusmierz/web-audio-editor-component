Web Audio Editor Component - Dev Demo

This page uses the bundle generated by npm run dev (dist/bundle.js).

The editor should appear below. Use the "Load Audio" button inside the component to load a file.

Instructions:
  1. Run npm install if you haven't already.
  2. Run npm run dev in your terminal.
  3. Open this file (demo.html) in your browser.
  4. Use the "Load Audio" button in the editor to select an audio file (e.g., MP3, WAV).

Check the browser's developer console for logs.
